What AstroPlan changes at import

AstroPlan does not alter these source files. Instead, at catalog import time it applies a documented set of corrections and additions — cross- referencing other public catalogs, fixing a small number of errors, and supplementing common names. The complete, structured list of every change is published here:

modifications.json

In summary, the modifications are:

  • Caldwell catalogue numbers — OpenNGC does not include Caldwell numbers; 105 are added by cross-reference, including two (C37, C50) redirected to the surviving object where OpenNGC's own data marks the traditional designation as a duplicate.
  • Messier 102 — OpenNGC marks M102 as a duplicate of M101 with no data of its own. Modern consensus (SIMBAD/NED) identifies it as NGC 5866; that assignment is added back.
  • One type-code correction — NGC 6611 (M16) is tagged as a nebula in OpenNGC; it is the star cluster (IC 4703 is the nebula it illuminates), matching SIMBAD/NED classification.
  • Common names — 45 Messier objects, 28 Caldwell objects, and 29 other objects gain common names absent from OpenNGC (sourced from Wikipedia's Messier/Caldwell tables, or well-established astrophotography usage), appended after any name OpenNGC already lists.
  • Herschel 400 program — encoded as a set of 402 NGC numbers (the program has 400 entries; three are traditionally listed under a pair of numbers for the same object). Six further entries are redirected to their surviving non-duplicate counterpart where OpenNGC marks the traditional number as a duplicate.
  • Cross-catalog merges — a small number of Sharpless and RCW entries (6 total) are identified as the same physical object as an existing NGC/IC entry and merged rather than listed twice, where the automatic cross-reference detection didn't catch it.

Other catalogs used

AstroPlan also bundles data from the Sharpless, RCW, Lynds (LDN), and Barnard catalogues, obtained via the VizieR catalogue access tool, CDS, Strasbourg, and Digitized Sky Survey imagery via the CDS hips2fits service. Full citations for each are listed in the app's About screen.
